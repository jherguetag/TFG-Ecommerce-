# E-Commerce Tecnológico - Proyecto DAM - Javier Hergueta Guerra

 Mi trabajo final del grado de Desarrollo de Aplicaciones Multiplataforma de 2º DAM consiste en una plataforma completa de comercio electrónico centrada en productos tecnológicos.

## Tecnologías utilizadas
**Backend:** Java 17 + Spring Boot (MVC)

**Persistencia:** MySQL + Spring Data JPA (Hibernate)

**Frontend:** HTML5, CSS3, JavaScript y Thymeleaf

**Diseño:** Bootstrap 5 + SweetAlert2 (UX)

## Funcionalidades
**CRUD Completo:** Gestión total de productos por parte del administrador

**Seguridad:** Control de acceso basado en roles (Admin/Cliente)

**Base de Datos Normalizada:** Relación entre Productos y Categorías.

**Diseño Responsivo:** Adaptado a móviles, tablets y escritorio
